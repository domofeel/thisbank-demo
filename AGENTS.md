# AGENTS.md

This is a simple React landing page with animations, sliders, and designer-driven visuals. The project is maintained mostly by designers, so keep actions small, clear, and predictable.

## Do Only What Was Asked

If the user only asks to run the project, open localhost, or provide a URL:

- Do not inspect the whole project.
- Do not edit files.
- Do not fix lint, formatting, dependency, or runtime issues unless explicitly asked.
- Read only the minimum needed files, such as `package.json` and lockfiles.
- Start the dev server and provide the actual URL.
- If startup fails, report the key error and ask before changing anything.

## Editing Rules

- Before editing, briefly say what will be changed.
- Keep changes small and local.
- Do not refactor unrelated code.
- Do not rename files, components, classes, assets, or CSS selectors unless asked.
- Do not add libraries or change package versions without approval.

## Protect The Design

- Preserve layout, spacing, typography, colors, animations, slider behavior, hover effects, and responsive behavior unless asked to change them.
- Use the existing styling approach.
- Do not remove, move, rename, compress, or replace images, videos, fonts, SVGs, or animation assets unless asked.
- Treat sliders, carousels, and scroll animations as fragile; make the smallest possible change.

## Communication

- Keep explanations short and practical.
- Avoid heavy technical jargon.
- If something fails, show the key error and explain it simply.
