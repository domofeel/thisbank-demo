# ThisBank

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

The production build is written to `dist`.

## Deploying To Vercel

1. Import the project in Vercel.
2. Use the Vite framework preset.
3. Build command: `npm run build`.
4. Output directory: `dist`.
5. Deploy.
